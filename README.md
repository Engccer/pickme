# 🎯 Pick Me - 무작위 학생 선발 도구

선생님들을 위한 재미있고 공정한 학생 선발 웹 애플리케이션입니다.

## 📌 소개

Pick Me는 학급에서 학생을 무작위로 선발할 때 사용하는 도구입니다. 행사 도우미, 청소 당번, 발표자 등을 선정할 때 공정하고 재미있게 뽑을 수 있습니다.

### 주요 특징
- ✨ **3가지 재미있는 테마** - 룰렛, 로또, 낚시 테마로 선발 과정이 즐겁습니다
- 🎲 **완전 무작위 선발** - 공정한 랜덤 알고리즘으로 선발합니다
- 👥 **성별 조건 설정** - 필요시 성별별 인원을 지정할 수 있습니다
- 🚫 **제외 기능** - 특정 학생을 선발 대상에서 제외할 수 있습니다
- 🎁 **비밀 선발 기능** - CSV 파일에서 미리 지정한 학생을 우선 선발할 수 있습니다
- 💾 **결과 저장** - 선발 결과를 텍스트 파일로 저장할 수 있습니다
- ♿ **접근성** - 스크린 리더를 지원하여 시각장애인도 사용 가능합니다

## 🚀 사용 방법

### 1단계: 학생 명단 준비

먼저 학생 명단을 CSV 파일로 준비해야 합니다.

**CSV 파일 형식:**
```csv
학년,반,번호,이름,성별,비밀선발
1,1,1,김철수,남,0
1,1,2,이영희,여,0
1,1,3,박지민,여,1
```

**각 열의 의미:**
- **학년**: 학생의 학년 (예: 1, 2, 3)
- **반**: 학생의 반 (예: 1, 2, 3)
- **번호**: 학생의 번호 (예: 1, 2, 3)
- **이름**: 학생의 이름
- **성별**: 남 또는 여
- **비밀선발**: 우선 선발할 학생이면 1, 아니면 0

**팁:** 엑셀에서 작성 후 "CSV (쉼표로 분리)"로 저장하면 됩니다.

### 2단계: 앱 실행

1. `index.html` 파일을 웹 브라우저로 엽니다
2. Chrome, Edge, Firefox 등 최신 브라우저 사용을 권장합니다

### 3단계: 학생 명단 불러오기

1. "📁 csv 파일을 선택하세요" 버튼을 클릭합니다
2. 준비한 CSV 파일을 선택합니다
3. 학생 수가 표시되면 "다음" 버튼을 클릭합니다

### 4단계: 선발 조건 설정

1. **총 선발 인원** 입력 (필수)
   - 뽑을 학생 수를 입력합니다
   - 예: 3명을 뽑으려면 "3" 입력

2. **성별 조건** (선택사항)
   - 성별 조건을 사용하려면 체크박스를 선택합니다
   - 여학생과 남학생 각각 몇 명씩 뽑을지 입력합니다
   - 예: 여학생 2명, 남학생 1명 (총 3명)

3. **선발 목적** (선택사항)
   - 선발 목적을 입력하면 결과 화면에 축하 메시지가 표시됩니다
   - 예: "행사 도우미", "청소 당번" 등

4. "다음" 버튼 클릭

### 5단계: 제외할 학생 선택

1. 선발 대상에서 제외할 학생이 있다면 체크박스를 선택합니다
2. 체크된 학생은 선발되지 않습니다
3. "다음" 버튼 클릭

### 6단계: 테마 선택

1. 원하는 테마를 선택합니다:
   - **🎰 룰렛**: 회전하는 룰렛으로 선발
   - **🎱 로또**: 공이 튀어나오는 연출
   - **🎣 낚시**: 낚시로 학생 선발

2. "🎯 선발 시작!" 버튼 클릭

### 7단계: 선발 진행

- 선택한 테마의 3D 애니메이션이 재생됩니다
- 배경 음악과 효과음이 함께 재생됩니다
- 선발되는 학생이 실시간으로 화면에 표시됩니다
- **일시 정지**: 언제든지 "⏸ 일시 정지" 버튼으로 멈출 수 있습니다
  - **계속하기**: 선발을 이어서 진행합니다
  - **처음으로**: 선발을 취소하고 처음부터 다시 시작합니다

### 8단계: 결과 확인

1. 선발이 완료되면 결과 화면이 표시됩니다
2. 선발 목적을 입력했다면 축하 메시지와 GIF 애니메이션이 표시됩니다
3. 선발된 학생 목록이 순서대로 표시됩니다

### 9단계: 결과 저장

- "💾 결과 저장" 버튼을 클릭하면 텍스트 파일로 저장됩니다
- 파일명 형식: `yymmdd_hhmmss_선발결과.txt`
- 예: `250123_143025_선발결과.txt` (2025년 1월 23일 14시 30분 25초)
- 파일은 브라우저의 다운로드 폴더에 저장됩니다

## 💡 주요 기능 상세 설명

### 비밀 선발 기능

CSV 파일에서 "비밀선발" 열에 1을 입력한 학생은 우선적으로 선발됩니다.

**사용 예시:**
- 특정 학생에게 기회를 주고 싶을 때
- 순번 또는 역할 순서를 은밀하게 지정하고 싶을 때

```csv
학년,반,번호,이름,성별,비밀선발
1,1,3,박지민,여,1
```
위 예시에서 박지민 학생은 무작위 선발보다 우선하여 선발됩니다.

### 동명이인 처리

같은 이름의 학생이 여러 명 있을 경우, 자동으로 구분하여 표시합니다:
- 같은 반: "김철수 (3번)"
- 같은 학년 다른 반: "김철수 (2반 5번)"
- 다른 학년: "김철수 (2학년 1반 7번)"

### 성별 조건 선발

성별 조건을 사용하면:
1. 비밀 선발 학생 중 성별 조건에 맞는 학생을 먼저 선발
2. 부족한 인원은 무작위로 선발

**예시:**
- 총 5명 선발 (여학생 3명, 남학생 2명)
- 비밀 선발 학생: 여학생 1명, 남학생 1명
- 결과: 비밀 선발 2명 + 무작위 여학생 2명 + 무작위 남학생 1명

## 🐛 문제 해결

### CSV 파일이 안 읽혀요
- 파일이 UTF-8 인코딩인지 확인하세요
- 엑셀에서 "CSV UTF-8 (쉼표로 분리)"로 저장하세요
- 첫 번째 줄이 헤더(학년,반,번호,이름,성별,비밀선발)인지 확인하세요

### 소리가 안 나요
- 브라우저가 음소거 상태가 아닌지 확인하세요
- 페이지에서 한 번 클릭해야 소리가 재생됩니다 (브라우저 정책)
- Chrome의 경우 사이트 설정에서 소리 허용이 되어있는지 확인하세요

### 애니메이션이 느려요
- 컴퓨터 성능에 따라 속도가 다를 수 있습니다
- 다른 프로그램을 종료하고 다시 시도해보세요
- 브라우저를 최신 버전으로 업데이트하세요

### 결과가 저장되지 않아요
- 브라우저의 다운로드 설정을 확인하세요
- 팝업 차단이 되어있지 않은지 확인하세요
- 브라우저 보안 정책상 다운로드 폴더에만 저장 가능합니다
